# Build a Complete Trading Academy Learning Platform

Create a production-ready Progressive Web Application (PWA) with Android compatibility through Capacitor.

The platform is a Trading Education Ecosystem with three roles:

1. Admin
2. Faculty
3. Student

The application must be responsive and optimized for:

* Mobile
* Tablet
* Desktop

Use modern UI/UX with a professional trading academy theme.

---

# Technology Stack

Frontend:

* React
* TypeScript
* Vite
* Tailwind CSS
* React Router

Backend:

* Supabase (Database)

Authentication:

* Firebase Authentication

Notifications:

* Firebase Cloud Messaging (FCM)

Documents:

* Google Drive

Videos:

* YouTube

Android Packaging:

* Capacitor

---

# Authentication System

Use Firebase Authentication.

Features:

* Email + Password Login
* Email Verification
* Password Reset
* Session Management

Signup Allowed:

* Student
* Faculty

Admin Accounts:

* Created directly in database
* No public admin registration

Login Available:

* Student
* Faculty
* Admin

After login, automatically redirect users according to their role.

Student → Student Dashboard

Faculty → Faculty Dashboard

Admin → Admin Dashboard

---

# User Roles

## Student

Permissions:

* View Assigned Courses
* Watch Recorded Lectures
* Join Live Classes
* Access Notes
* View Certificates
* Receive Notifications
* Update Profile

Cannot:

* Upload Content
* Manage Courses
* Manage Users

---

## Faculty

Permissions:

* Upload Recorded Lecture
* Create Live Class
* Upload Notes
* Manage Own Content
* View Own Courses

Cannot:

* Manage Students
* Manage Faculty
* Manage Certificates
* Access Admin Features

---

## Admin

Permissions:

* Manage Students
* Manage Faculty
* Manage Courses
* Manage Batches
* Assign Courses
* Approve Accounts
* Reject Accounts
* Block Users
* Unblock Users
* Manage Notifications
* Manage Certificates
* View Activity Logs

Workflow:

→ Select Interested Course

→ Admin Review

→ Admin Assign Course

→ Student Gets Access

# Recorded Lectures

Faculty can create lectures.

Fields:

* Lecture Title
* Description
* Course
* YouTube Video ID
* Thumbnail

Store metadata in Supabase.

Videos must be displayed using a custom video player interface.

Hide:

* Share Button
* Download Button
* External YouTube Navigation

Use YouTube Embedded Player API.

---

# Live Classes

Faculty can create live classes.

Fields:

* Class Title
* Description

* Time
* Course
